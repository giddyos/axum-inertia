# Askama root example

This focused example uses Askama for the application-wide HTML document and Vue for the Inertia page component. Askama compiles `templates/app.html` as part of this Rust crate, so the server neither copies nor reads the template at runtime.

The root template is rendered only for an initial HTML response. Vue continues to render the `Home` page, and subsequent requests carrying `X-Inertia: true` receive the normal JSON page object instead of another root document. The same root integration also works with React or Svelte page components.

`inertia.assets`, `inertia.head`, and `inertia.mount` are trusted markup generated by `inertia-axum`, so those three values use Askama's `safe` filter. Normal application values such as the app name, description, locale, and social-image URL retain Askama's default HTML escaping. Do not apply `safe` to user or application content.

Build the frontend and run the server:

```sh
cd examples/axum-askama/frontend
npm install
npm run build
cd ..
cargo run -p axum-askama
```

The `askama` feature is optional. Applications using the built-in root or the startup-compiled marker template do not need it.
